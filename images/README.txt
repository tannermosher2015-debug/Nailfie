NAILFIE STUDIO — image drop-in guide
====================================

Every photo slot on the site is a styled placeholder that AUTO-REPLACES itself the
moment a matching file exists here. Drop the JPGs in this folder with these exact
names — no code changes needed.

IMPORTANT: use the CLEAN original photo from each @nailfiehi post — NOT a screenshot
with the Instagram like/comment overlay on it, and one nail set per file (don't use a
stitched collage). Crop roughly to the ratio shown.

The gallery captions are written to match these specific real sets, so save the photo
that matches each slot (or edit the caption in index.html if you swap the design):

  hero.jpg      4:5 portrait (~1080×1350)  Your best/standout sculpted set, tight macro. Face of the site.
  maylei.jpg    3:4 portrait (~1080×1440)  Maylei at her station / a portrait of her.

  work-01.jpg   4:5   Black & white hand-painted swirl (the green pop-art accent set)
  work-02.jpg   4:6   Lilac/purple ombré with pressed flowers + butterflies (taller crop)
  work-03.jpg   4:5   Pink & white French tip with silver glitter
  work-04.jpg   4:5   Milky / jelly almond (the sheer blue set)
  work-05.jpg   4:5   Sage-green swirl French with black linework
  work-06.jpg   4:6   Holiday red — bows / holly / snowflakes (taller crop)

  og-cover.jpg  1200×630 landscape  Social-share preview (optional)
